# Stock Scanner (주식 스캐너) — PRD v1.1

> **1차 MVP** | 목표 기간: 2–4주 | 기술 스택: React + FastAPI  
> 타겟: 소규모 투자 커뮤니티 | UI: 한국어 우선 | 데이터: 장 마감 종가 기준

---

## 1. 프로젝트 개요

### 비전

장 마감 종가 기준으로 미국/한국 시장의 종합 지수, 테마별 추세, 개별 종목 데이터를 수집하고, AI 분석을 통해 투자 인사이트를 제공하는 웹 기반 주식 스캐너. 소규모 투자 커뮤니티를 위한 공유용 툴로서, 빠른 시장 파악과 테마/종목 발굴을 목표로 함.

### 핵심 문제 정의

- 시장 전체를 한눈에 파악하기 어려움 — 지수, 테마, 개별 종목이 분산되어 있음
- 테마/섹터별 추세 파악에 시간이 많이 소요됨 — 수동 분석 한계
- 기술적 지표(RSI, MACD, 이평선 등) 기반 스크리닝이 불편함
- 수집된 데이터를 종합적으로 분석해주는 도구 부족

### 목표 사용자

소규모 투자 커뮤니티 멤버 (5–50명 규모). 중급 이상 개인 투자자로, 기술적 분석과 테마 투자에 관심이 있으며, 장 마감 후 빠른 시장 요약과 종목 발굴을 원하는 사람들.

## 2. 데이터 전략

### 데이터 수집 원칙

- **기준 시점**: 장 마감 종가 (EOD, End of Day)
- **수집 주기**: 매일 1회 (US 장 마감 후 / KR 장 마감 후 각각 실행)
- **저장소**: PostgreSQL + Redis 캐시

### API 소스 구성

| 소스 | 커버리지 | 제공 데이터 | 비고 |
|------|----------|------------|------|
| Yahoo Finance | US 시장 전체 | 종가, 거래량, 재무지표, 차트 | yfinance 라이브러리 활용 |
| KRX (한국거래소) | KOSPI / KOSDAQ | 종가, 시총, PER, PBR | pykrx / 공공데이터 API |
| FRED | 매크로 지표 | 금리, 환율, 채권수익률 | 무료 API Key |
| CoinGecko | 코인 시장 | BTC, ETH 등 주요 코인 가격 | 무료 티어 충분 |
| News API / RSS | 뉴스 | 시장 관련 뉴스 수집 | 1차: RSS 크롤링 → 2차: 유료 API |

### 기술적 지표 산출 (백엔드)

yfinance/pykrx로 수집한 종가 데이터를 기반으로 백엔드에서 다음 지표를 산출:

- RSI (14일), MACD (12, 26, 9), 볼린저 밴드
- 60일 / 120일 / 240일 이동평균선 및 현재가 대비 괴리%
- 52주 고점 대비 하락율 (%)
- 거래량 비율 (20일 평균 대비)
- 이평선 터치 판별 (종가가 특정 이평선 ±2% 이내인지 체크)

### 3.2 시장 상세 페이지

각 시장별 전용 페이지. 1차 MVP에서는 다음 4개 시장을 지원:

- NASDAQ (US)
- S&P 500 (US)
- KOSPI (KR)
- KOSDAQ (KR)

#### 섹션 A: 시장 지수 차트

- 일봉/주봉/월봉/년봉 토글 가능한 캔들스틱 차트
- 주요 이평선 (20/60/120/240일) 오버레이
- 거래량 바 차트 함께 표시
- 차트 라이브러리: **Lightweight Charts** (TradingView 오픈소스) 추천

#### 섹션 B: 테마별 추세 표

해당 시장 내 테마/섹터별 강도를 시각적으로 표현:

- 테마 분류 기준: GICS 섹터 + 커스텀 테마태그 (AI/반도체/2차전지/방산 등)
- 표시 항목: 테마명, 종목수, 평균 상승률, 평균 RSI, 거래량 변화율, 추세 강도(색상 코드)
- 정렬: 추세 강도순 (강한 테마 → 약한 테마)
- 클릭 시 해당 테마 종목 목록으로 필터링

#### 섹션 C: 시장 브레드스 (Market Breadth)

시장 건강도를 파악하는 지표 모음:

- 상승/하락 종목 비율 (파이 차트 또는 게이지)
- 52주 신고가 vs 신저가 종목 수
- 20일/60일/120일/240일 이평선 위 종목 비율
- RSI 과매수/과매도 종목 비율

#### 섹션 D: 뉴스 요약

- RSS/News API로 수집한 시장 관련 뉴스 상위 10개
- 각 뉴스에 AI 요약 (1줄) + 영향 태그 (금리/인플레/실적/정책 등)
- 클릭 시 원문 링크 연결
- 1차 MVP: RSS 크롤링 (무료) → 후속: News API 유료 업그레이드

#### 섹션 E: 종목 데이터 테이블

핵심 기능. 검색, 필터, 정렬이 가능한 종목 리스트:

| 컬럼 | 설명 | 비고 |
|------|------|------|
| 티커 (인베스팅 링크) | 클릭 시 Investing.com 종목 페이지로 이동 | 외부 링크 |
| 종목명 | 한글/영문 종목명 | 검색 가능 |
| 테마 | 소속 테마/섹터 | 필터 가능 |
| 추세 | 상승/횡보합/하락 판별 | 색상 코드 |
| RSI (14) | 14일 RSI 값 | 과매수(<30)/과매도(>70) 표시 |
| MACD | MACD 신호 (0선 위/아래) | 골든/데드크로스 표시 |
| 현재가 | 종가 기준 | |
| 상승률 (%) | 전일 대비 | 색상 코드 |
| 거래량 비율 | 20일 평균 대비 배수 | 폭증 시 하이라이트 |
| 고점대비 -% | 52주 고점 대비 하락율 | |
| PER / PBR | 재무 지표 | 업종 평균 비교 가능 |
| 시총 | 시가총액 | 단위: 억원 / M$ |
| 업사이드 (%) | 목표가 기반 상승 여력 | 컨센서스 목표가 기준 |
| 60/120/240일선 괴리% | 각 이평선 대비 현재가 차이 | 양수/음수 색상 |
| 이평선 터치 | 종가가 해당 이평선 ±2% 이내 | 변동성 기준 자동 조정 |

#### 스크리닝 프리셋 (빠른 필터)

사전 정의된 필터 조합을 버튼 하나로 적용:

- **과매도 반등 후보**: RSI < 30 + 거래량 비율 > 2배 + 상승률 > 0%
- **골든크로스**: MACD 골든크로스 + 60일선 위
- **이평선 지지**: 120일선 터치 + RSI 40–60 중립구간
- **고점 급락**: 고점대비 -30% 이상 + PER < 업종평균
- **거래량 폭발**: 거래량 비율 > 5배 (오늘 급등 종목)

#### 섹션 F: 히트맵 뷰

시장 전체 종목을 섹터/테마별로 시각화:

- 블록 크기: 시총 기준
- 색상: 상승률 기준 (빨간색 → 초록색 그라데이션)
- 호버: 종목명, 상승률, 시총 표시
- 클릭 시 종목 상세 모달
- 티어: 1차에서는 D3.js treemap 활용

#### 섹션 G: 섹터 로테이션 차트

- X축: 상대적 성과 (vs 시장평균), Y축: 모멘텀 (최근 1개월 변화량)
- 4사분면: Leading / Weakening / Lagging / Improving
- 버블 크기: 해당 섹터 시총
- 주기적 업데이트로 자금 흐름 추적

### 3.3 AI 분석 리포트

#### 매일 자동 생성 시장 요약 리포트

##### 리포트 구성

1. **시장 총평**: 오늘 시장 흐름 요약 (2–3단락)
2. **테마 분석**: 강세/약세 테마 해설 + 근거
3. **주목 종목**: AI가 선별한 관심 종목 5–10개 (근거 포함)
4. **리스크 요인**: 금리/환율/지정학적 리스크 요약
5. **뉴스 요약**: 주요 뉴스가 시장에 미치는 영향 분석

##### AI 분석 입력 데이터

- 종합 지수 변동 + 기술적 지표
- 테마별 성과 데이터
- 시장 브레드스 지표
- 수집된 뉴스 텍스트
- 개별 종목 스캐닝 결과
- 추적기 도출 결과

##### AI 모델

- 1차 MVP: **Claude API (Anthropic)** — 시장 데이터를 구조화한 프롬프트로 전달, 분석 리포트 생성
- 대안: OpenAI GPT-4o 또는 로컬 LLM
- 프롬프트 템플릿을 버전 관리하여 분석 품질 지속 개선

##### 면책 조항

> 모든 AI 분석 결과에는 "본 분석은 AI에 의해 생성되었으며, 투자 조언이 아닙니다. 투자 판단은 본인 책임 하에 이루어져야 합니다." 문구를 필수 표시.

### 3.5 개인 추적기 (Momentum Tracker)

시장 전체 종목을 매일 자동으로 스캔하여, 다양한 관점의 분석을 통합하고 매매 후보 종목을 능동적으로 도출하는 시스템. 웹 페이지 내 전용 탭으로 제공하며, 각 전략별 최대 20개 종목을 도출함.

#### 전략 1: 상승추세 눌림 포착 (Pullback in Uptrend)

상승 추세가 유지되는 종목들 중 장기 이평선까지 눌림(pullback)이 발생한 종목을 포착하고, 복합 지표 분석을 통해 반등 가능성이 높은 종목을 선별.

##### 필터링 조건 (1차 스크리닝)

| 조건 | 기준 | 설명 |
|------|------|------|
| 추세 판별 | 240일선 상향 + 현재가 > 240일선 | 장기 상승추세 확인 |
| 눌림 감지 | 현재가가 60/120/240일선 ±2% 이내 | 이평선 터치 구간 진입 |
| RSI 구간 | RSI 30–50 (과매도 해소 직전) | 가격 눌림이 충분히 진행된 상태 |
| MACD | MACD 히스토그램 감소 반전 조짐 | 하락 모멘텀 둔화 신호 |
| 거래량 | 하락 시 거래량 감소 (평균 이하) | 매도 압력 약화 확인 |

##### 복합 스코어링 (2차 랭킹)

1차 스크리닝을 통과한 종목에 대해 다음 관점의 점수를 종합하여 순위 도출:

| 분석 관점 | 세부 항목 | 배점 기준 | 비중 |
|-----------|----------|----------|------|
| 차트 (30%) | 이평선 배열, 지지/저항, 추세선 | 이평선 정배열 + 지지선 근접 시 고점 | **30%** |
| 기술지표 (25%) | RSI, MACD, 볼린저, 거래량 | RSI 반전 + MACD 골든 조짐 시 고점 | **25%** |
| 재무 (20%) | PER, PBR, 실적성장률, ROE | 업종 평균 대비 저평가 + 실적성장 시 고점 | **20%** |
| 테마 (15%) | 테마 강도, 섹터 모멘텀 | 강세 테마 소속 + 섹터 Leading 시 고점 | **15%** |
| 수급 (10%) | 고점대비 하락률, 업사이드 | 하락폭 클수록 + 업사이드 높을수록 고점 | **10%** |

#### 전략 2: 하락추세 이탈 포착 (Trend Reversal)

하락 추세에 있던 종목들 중 과매도 구간이거나 하락추세를 이탈하는 신호를 보이는 종목을 포착.

##### 필터링 조건

| 유형 | 조건 | 설명 |
|------|------|------|
| A. 과매도 반등 | RSI < 30 + 고점대비 -30% 이상 하락 | 극단적 과매도에서 반등 기대 |
| B. 추세선 돌파 | 종가가 60일선 상향 돌파 + 거래량 폭증 | 하락추세 이탈 초기 신호 |
| C. MACD 반전 | MACD 0선 상향 돌파 + 골든크로스 | 모멘텀 전환 확인 |
| D. 볼린저 수축 후 확장 | 볼린저 밴드폭 수축 후 상방 돌파 | 변동성 확대 구간 진입 |

#### 추적기 UI 구성

- 전용 페이지: 상단 탭으로 "상승추세 눌림" / "하락추세 이탈" 전환
- 각 종목 카드: 종합점수 + 각 관점별 점수 시각화 (레이더 차트 또는 점수 바)
- 미니 차트: 각 종목 카드에 최근 60일 캔들차트 + 이평선 표시
- 클릭 시 상세 모달: 전체 차트 + 분석 근거 + AI 요약
- 테마/시장별 필터: 특정 테마나 시장만 볼 수 있는 필터
- 최대 도출 종목: 각 전략별 20개 이하, 종합점수 순으로 정렬

#### AI 연동

- 추적기 도출 종목을 AI 분석 리포트에 반영 ("오늘의 눌림 포착 종목" 섹션)
- 각 종목별 1줄 AI 코멘트 자동 생성 (예: "반도체 강세 테마 + 120일선 지지 확인 + 거래량 감소형 눌림")
- 텔레그램 발송 시 `#추적기` 채널에 별도 발송

#### 데이터 파이프라인 위치

기존 데이터 수집 파이프라인 뒤에 추가 단계로 실행:

① 종가/지표 수집 → ② 기술지표 산출 → ③ 테마 분류 → **④ 추적기 스크리닝** → **⑤ 복합 스코어링** → ⑥ AI 분석 → ⑦ 발송

### 5.1 시스템 구성

| 계층 | 기술 | 설명 |

| 스케줄러 | APScheduler / Celery | 매일 장 마감 후 데이터 수집 |
| AI | Claude API (Anthropic) | 시장 분석 리포트 생성 |
| 알림 | Telegram Bot API | 커뮤니티 리포트 발송 |
| 배포 | Docker + VPS (or Railway) | 초기에는 단일 서버 |

### 5.2 데이터 파이프라인 (Updated)

매일 장 마감 후 자동 실행되는 파이프라인 (추적기 단계 포함):

1. **데이터 수집**: yfinance + pykrx로 종가/거래량/재무 데이터 수집
2. **지표 산출**: RSI, MACD, 이평선, 볼린저, 브레드스 등 기술적 지표 계산
3. **테마 분류**: 종목별 테마 매핑 + 테마 성과 집계
4. **추적기 스크리닝**: 상승추세 눌림 / 하락추세 이탈 필터링
5. **복합 스코어링**: 차트/지표/재무/테마/수급 점수 산출 + 순위 도출
6. **뉴스 수집**: RSS 크롤링 + AI 요약
7. **AI 분석**: 수집된 데이터 + 추적기 결과를 LLM에 전달하여 리포트 생성
8. **텔레그램 발송**: 분석 결과 + 추적기 종목을 포맷팅하여 채널별 발송
